Below are a few APIs that we will be utilizing for our project.

Mapbox GL JS creates map object. Creating a map object allows us to modify the map and trigger events based on user's location.

'navigator.geolocation'

is the browser's built-in interface to obtain user's location

when used with methods like 'getCurrentPosition(...)', you can retrieve current coordinates of a user.
